About Me

Hi there! I'm a passionate data analyst who loves diving into data and turning it into easy-to-understand insights. My expertise lies in SQL Server, cleaning up messy data, and creating eye-catching visualizations.

What I do

  • SQL Server Wizardry: I use SQL Server to write complex queries and make sense of large datasets.
  • Data Cleaning Expert: I transform raw data into clean, organized, and reliable information.
  • Visual Storyteller: With tools like Matplotlib and Tableau, I create engaging charts and dashboards that make data come alive.

Projects I'm Proud Of

  • Housing Data Cleanup: I cleaned and organized large sets of housing data with advanced SQL techniques, making it ready for analysis and visualization.
  • COVID-19 Data Exploration: I analyzed global COVID-19 data using SQL Server, uncovering important trends and patterns in the virus's spread and impact.
  • NFT Trading Card Game Analysis: I gathered and visualized data from the "Gods Unchained" trading card game using Pandas and Matplotlib, revealing interesting insights from the game.

My Approach

I believe every piece of data has a story to tell. Whether it's understanding pandemic trends, optimizing real estate data, or analyzing digital collectibles, I approach each project with curiosity and dedication. I enjoy making complex data simple and meaningful for everyone.
